(static) execute_array_hooks(hook_name, params, acc_index) → {object}

Source:
  • client/arch/hooks.js, line 103

Execute all hooks attached to the given name in order. Hooks are executed sequentially such that the first hook's return value is passed into the second and so on.

Parameters:
Name Type Description
hook_name string

the name of the hook
params Array

parameters to pass to the hooks
acc_index number

the index in params in which the result of the hooks must be saved, if no hooks exist for the name, then params[acc_index] is returned.
Returns:

returns the result of the last hook.
